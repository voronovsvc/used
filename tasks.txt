Как закончишь задания:
1. Добавить сеттер для установки свойста layout в классе View, 
таким образом сможем использовать разные лайауты на разных страницах, если потребуется.



2. Сделать возможность добавлять дополнительные скрипты в контроллере, перед рендером:
- Добавить в класс View свойство js, по-умолчанию пустой массив.

- Добавить метод addJs(), который в качестве параметра будет принимать путь к js файлу в папке js
и будет сохранять его в массив приватного свойста js.

- Добавить метод getJs(), который будет будет возвращать строку из тегов <script> с подключенными файлами из свойста js
(Если в массиве много файлов, надо вывести все)
- Вывести результат вызыва getJs() (если он не пустой) в layout.tpl в секцию <head> сразу под уже подключенными скриптами

3. Аналогичным образом сделать возможность подключать дополнительные стили.
Используй  приватное свойство css, методы addCss() и getCss(). Логика работы такая-же как со скриптами

Чтоб в результате в методах контроллеров можно было делать так:
$this->view->addJs('/js/script1.js');
$this->view->addCss('/css/style1.css');
Если такие файлы существуют они должны подключиться только на этой странице
