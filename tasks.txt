0. Создай базу данных для проекта.
1. Создай в корне проекта папку config
2. В папке config файл config.php
3. В этом файле нужно объявить константы:
define('DB_SERVER', 'localhost');
define('DB_NAME', ''); // Сюда впиши имя своей базы
define('DB_USER', 'root'); // Имя пользователя базы данных
define('DB_PASSWD', ''); // Пароль от базы
4. Подключи этот файл в index.php

Класс для работы с базой:

1. Почитай что такое класс-синглтон.
2. Создай в папке app/core файл db.php, в нем синглтон-класс DB.
3. Конструктор этого класса должен приниметь 4 параметра- $user, $pass, $dbname, $server
Параметр $server должен иметь значение по-умолчанию: 'localhost'

Это будет имя пользователя базы данных, пароль, имя базы данных и сервер
То есть чтоб вот так работало (это просто пример): 
$db = new DB($server, $user, $pass, $dbname);
